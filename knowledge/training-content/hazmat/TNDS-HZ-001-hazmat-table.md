# Hazardous Materials Table

## Learning Objectives
- Explain the purpose of the Hazardous Materials Table and where to find it
- Identify each column in the Hazmat Table and describe what information it provides
- Look up a material in the Hazmat Table and extract the required shipping information
- List and describe the nine DOT hazard classes
- Explain the three packing groups and what they indicate about danger severity
- Recognize common mistakes when using the Hazmat Table

## Section 1: Purpose of the Hazmat Table

The Hazardous Materials Table, found at 49 CFR 172.101, is the single most important reference in the entire HMR. It is the master list of every material regulated as hazardous for transportation purposes. When a shipper needs to determine how to classify, package, mark, label, and document a hazardous material, the Hazmat Table is where they start. When a carrier needs to verify that a shipment was properly prepared, the Hazmat Table is where they look. When an inspector finds a discrepancy, the Hazmat Table is the standard they measure against.

The table contains over 3,000 entries organized in alphabetical order by proper shipping name. Each entry provides all the regulatory information needed to ship that specific material: its hazard class, identification number, packing group, authorized packagings, quantity limits, and special conditions. Think of it as a detailed instruction manual for every hazardous material that moves in commerce.

For fleet operators, familiarity with the Hazmat Table is essential. Even if you are not the one preparing shipments, you need to understand the table well enough to verify that the shipping papers, marks, labels, and placards on your vehicles match what the table requires. A mismatch between what the table says and what is on the truck is a violation — and potentially a safety hazard.

The Hazmat Table also connects directly to the Emergency Response Guidebook (ERG). The Yellow Pages of the ERG are organized by UN/NA identification number, and the Blue Pages are organized by material name — both of which come straight from the Hazmat Table. Being able to cross-reference between the Hazmat Table and the ERG is a core competency for anyone in hazmat transportation.

## Section 2: Column-by-Column Walkthrough

The Hazmat Table has 10 columns. Understanding what each column tells you is fundamental to using the table correctly.

### Column 1: Symbols

The first column may contain one or more letter symbols that provide special instructions or limitations for that entry. The most common symbols you will encounter are:

- **"+" (Plus sign)**: This material must be shipped under the exact proper shipping name and hazard class shown in the table, even if it could technically fit a different entry. The shipper has no flexibility.
- **"A"**: This material is subject to the HMR only when transported by aircraft. If you are hauling it by highway, the HMR may not apply (though other regulations might).
- **"D"**: This proper shipping name is appropriate for domestic transportation only. For international shipments, a different entry with a "UN" identification number must be used instead.
- **"G"**: This is a generic or "not otherwise specified" (n.o.s.) entry. The shipper must add a technical name in parentheses after the proper shipping name on shipping papers. For example, "Flammable liquid, n.o.s. (contains ethanol)."
- **"I"**: This entry is appropriate for international transportation only. A different entry must be used for domestic highway shipments.
- **"W"**: This material is subject to the HMR only when transported by water (vessel). Highway carriers generally will not encounter these entries.

**Practical Tip**: Most entries a highway fleet will encounter have no symbol or a "D" symbol. If you see a "G" symbol, check the shipping papers carefully — the technical name must be included.

### Column 2: Proper Shipping Name

This is the official, regulated name that must be used on shipping papers, marks, and labels. It is not a brand name, trade name, or common nickname. For example, the proper shipping name for gasoline is "Gasoline" — not "unleaded regular" or "87 octane." The proper shipping name for bleach is "Hypochlorite solution" — not "Clorox" or "bleach."

Proper shipping names are shown in **regular type** (roman) in the table. Italicized text in Column 2 is not part of the proper shipping name — it is a description or cross-reference to help you find the right entry. For example, you might see "Alcohol — see Ethanol" in italics, directing you to the correct entry.

Some proper shipping names include qualifiers like "flammable," "corrosive," "solution," or "solid" as part of the official name. These qualifiers are part of the required name and must appear on shipping papers.

### Column 3: Hazard Class or Division

This column tells you the hazard class and division number assigned to the material. The nine DOT hazard classes are covered in detail in Section 4 of this module. The hazard class drives the labeling, placarding, and compatibility requirements for the material.

Some materials have a primary hazard class and one or more subsidiary hazard classes. For example, a material might be primarily a Class 3 Flammable Liquid but also have a subsidiary hazard of Class 6.1 Toxic. The subsidiary hazards are listed in Column 6 (Special Provisions) or may require additional labels.

### Column 4: Identification Number

Every hazardous material is assigned a four-digit identification number preceded by either "UN" (United Nations — used internationally) or "NA" (North America — used only in the U.S. and Canada). Examples:

- Gasoline: **UN1203**
- Diesel Fuel: **NA1993**
- Acetone: **UN1090**
- Propane: **UN1075**

This number appears on shipping papers, package marks, placards (for bulk shipments), and in the ERG. It is the universal identifier that allows anyone — a driver, an inspector, a firefighter — to quickly determine what material they are dealing with.

### Column 5: Packing Group

The packing group indicates the degree of danger the material presents during transportation. There are three levels:

- **Packing Group I**: Great danger. These materials are the most hazardous within their class. They require the most robust packaging and the most restrictive handling.
- **Packing Group II**: Medium danger. These materials pose a moderate level of hazard.
- **Packing Group III**: Minor danger. These materials are the least hazardous within their class, though they are still regulated and require proper handling.

Not all hazard classes use packing groups. Classes 1 (Explosives), 2 (Gases), 5.2 (Organic Peroxides), 6.2 (Infectious Substances), and 7 (Radioactive) do not use packing groups — they have their own internal classification systems.

### Column 6: Special Provisions

This column lists numbers that correspond to special provisions in 49 CFR 172.102. These provisions modify the standard requirements for a particular material. For example, a special provision might exempt certain concentrations of a material from the HMR, require a specific packaging type, or mandate additional labeling. Always check the special provisions — they can significantly change what is required for a shipment.

### Columns 7 (Label Codes)

This column specifies which hazard labels must be applied to packages of the material. Label codes correspond to specific diamond-shaped labels (e.g., "3" requires a FLAMMABLE LIQUID label, "8" requires a CORROSIVE label). If multiple codes are listed, multiple labels are required.

### Columns 8A, 8B, 8C: Packaging Authorizations

These three sub-columns reference the specific sections of 49 CFR Part 173 that authorize packagings for the material:

- **8A — Exceptions**: References exceptions from packaging requirements (less restrictive rules for certain quantities or configurations).
- **8B — Non-bulk**: References the authorized non-bulk packagings (drums, boxes, cans — generally under 119 gallons for liquids or 882 pounds for solids).
- **8C — Bulk**: References the authorized bulk packagings (cargo tanks, portable tanks, intermediate bulk containers — over the non-bulk thresholds).

### Columns 9A, 9B: Quantity Limitations

- **9A — Passenger Aircraft/Rail**: The maximum quantity of the material allowed on a passenger-carrying aircraft or railcar. "Forbidden" means the material cannot travel on passenger conveyances at all.
- **9B — Cargo Aircraft Only**: The maximum quantity allowed on cargo-only aircraft.

These columns are primarily relevant for air transportation but can come into play when a fleet handles intermodal shipments.

### Columns 10A, 10B: Vessel Stowage

- **10A — Location**: Specifies stowage categories for ocean vessels (e.g., "A" for on deck or below deck, "B" for on deck only).
- **10B — Other**: Lists additional stowage codes with specific requirements.

These columns apply to water transportation and are generally not a concern for highway-only fleet operations.

## Section 3: How to Look Up a Material

Using the Hazmat Table effectively requires a systematic approach. Here is a step-by-step example using gasoline, one of the most commonly transported hazardous materials.

### Step 1: Find the Entry

Open 49 CFR 172.101 (or use an electronic version) and look up "Gasoline" alphabetically. You will find:

| Symbol | Proper Shipping Name | Hazard Class | ID Number | Packing Group |
|--------|---------------------|-------------|-----------|---------------|
| (none) | Gasoline | 3 | UN1203 | II |

### Step 2: Read the Hazard Class

Gasoline is **Class 3 — Flammable Liquid**. This means it has a flash point below 141 degrees Fahrenheit (60 degrees Celsius). This classification determines the labels on packages and the placards on vehicles.

### Step 3: Note the ID Number

**UN1203**. This number will appear on the shipping papers, on package marks, and on the orange panel of bulk placards. If a first responder arrives at a gasoline spill on the highway, they will look for UN1203 on the placard, then turn to Guide 128 in the ERG for response instructions.

### Step 4: Check the Packing Group

**Packing Group II — Medium danger**. This tells you gasoline requires mid-level packaging integrity. It is not the most volatile flammable liquid (that would be PG I), but it is more dangerous than PG III materials like diesel fuel.

### Step 5: Review Special Provisions

Check Column 6 for any special provision numbers, then look them up in 49 CFR 172.102 to see if any modify the standard requirements.

### Step 6: Identify Packaging Authorizations

Check Columns 8A, 8B, and 8C for the specific Part 173 sections that authorize packaging. For gasoline, non-bulk packaging authorizations reference 49 CFR 173.202, and bulk packaging authorizations reference 49 CFR 173.242.

### Another Example: Acetone

Look up "Acetone" and you find:

| Symbol | Proper Shipping Name | Hazard Class | ID Number | Packing Group |
|--------|---------------------|-------------|-----------|---------------|
| (none) | Acetone | 3 | UN1090 | II |

Acetone is also a Class 3 Flammable Liquid with Packing Group II, just like gasoline. But their specific packaging authorizations and special provisions may differ, so you must always check the full table entry for the specific material you are shipping — never assume two materials with the same class and packing group have identical requirements.

## Section 4: The Nine Hazard Classes

The DOT classifies hazardous materials into nine hazard classes. Every hazmat employee should be able to name these classes and recognize examples of materials in each one.

### Class 1: Explosives
Materials that can detonate or explode. Divided into six divisions based on blast hazard, projection hazard, and fire hazard.
- **Division 1.1**: Mass explosion hazard (dynamite, TNT)
- **Division 1.2**: Projection hazard (military munitions)
- **Division 1.3**: Fire hazard with minor blast (propellant, rocket motors)
- **Division 1.4**: Minor explosion hazard (small arms ammunition, fireworks)
- **Division 1.5**: Very insensitive explosives (blasting agents)
- **Division 1.6**: Extremely insensitive detonating articles

**Fleet Relevance**: Ammunition shipments, commercial fireworks, and blasting agents for construction and mining operations.

### Class 2: Gases
Materials that are gaseous at 68 degrees Fahrenheit (20 degrees Celsius) and 14.7 psi (standard atmospheric pressure). Three divisions:
- **Division 2.1 — Flammable Gas**: Burns in air (propane, butane, acetylene, hydrogen)
- **Division 2.2 — Non-Flammable, Non-Toxic Gas**: Compressed or liquefied gases that are not flammable or toxic (nitrogen, carbon dioxide, helium, compressed air)
- **Division 2.3 — Toxic Gas**: Poisonous by inhalation (chlorine, ammonia, hydrogen sulfide)

**Fleet Relevance**: Propane deliveries, welding gas cylinders, medical oxygen, refrigerant cylinders.

### Class 3: Flammable Liquids
Liquids with a flash point of 141 degrees Fahrenheit (60 degrees Celsius) or below. This is one of the most commonly transported hazard classes.
- Examples: Gasoline (UN1203), diesel fuel (NA1993), acetone (UN1090), ethanol (UN1170), paint (UN1263), adhesives (UN1133)

**Fleet Relevance**: Fuel hauling, chemical distribution, paint and coatings delivery, cleaning product distribution.

### Class 4: Flammable Solids
Three divisions of solid materials with fire hazards:
- **Division 4.1 — Flammable Solid**: Burns readily from friction or self-sustained combustion (matches, magnesium)
- **Division 4.2 — Spontaneously Combustible**: Ignites on contact with air (white phosphorus, certain metal powders)
- **Division 4.3 — Dangerous When Wet**: Reacts with water to produce flammable or toxic gas (sodium, calcium carbide, lithium)

**Fleet Relevance**: Metal powder shipments, match shipments, certain chemical raw materials.

### Class 5: Oxidizers and Organic Peroxides
- **Division 5.1 — Oxidizer**: Supplies oxygen to support combustion of other materials (ammonium nitrate, hydrogen peroxide, calcium hypochlorite)
- **Division 5.2 — Organic Peroxide**: Unstable compounds that can decompose explosively or burn rapidly (benzoyl peroxide, methyl ethyl ketone peroxide)

**Fleet Relevance**: Pool chemical deliveries (calcium hypochlorite), fertilizer shipments (ammonium nitrate), industrial chemical distribution.

### Class 6: Toxic and Infectious Substances
- **Division 6.1 — Toxic (Poison)**: Materials that can cause death or serious injury through ingestion, inhalation, or skin absorption (pesticides, cyanide, arsenic compounds)
- **Division 6.2 — Infectious Substance**: Materials that contain pathogens (medical waste, diagnostic specimens, biological cultures)

**Fleet Relevance**: Agricultural pesticide deliveries, medical waste transportation, pharmaceutical distribution.

### Class 7: Radioactive Materials
Materials that emit ionizing radiation above specified thresholds. Classified by transport index and surface radiation levels.
- Examples: Medical isotopes, industrial radiography sources, uranium ore, smoke detectors (exempt quantities)

**Fleet Relevance**: Medical isotope deliveries, construction density gauge transport.

### Class 8: Corrosive Materials
Materials that cause visible destruction or irreversible damage to skin tissue, or that corrode steel or aluminum at a specified rate.
- Examples: Sulfuric acid (UN1830), hydrochloric acid (UN1789), sodium hydroxide solution (UN1824), battery acid, bleach (hypochlorite solution)

**Fleet Relevance**: Chemical distribution, battery deliveries, cleaning product distribution, water treatment chemical deliveries.

### Class 9: Miscellaneous Hazardous Materials
Materials that present a hazard during transportation but do not fit into Classes 1-8. This is a catch-all class.
- Examples: Lithium batteries (UN3481, UN3091), dry ice/solid carbon dioxide (UN1845), environmentally hazardous substances, magnetized materials, elevated temperature materials

**Fleet Relevance**: Electronics shipments with lithium batteries, dry ice shipments for food or medical products, asphalt tankers (elevated temperature).

## Section 5: Packing Groups Explained

Packing groups tell you how dangerous a material is relative to others in the same hazard class. They drive packaging requirements — a PG I material must go in a stronger, more rigorously tested container than a PG III material.

### Packing Group I — Great Danger
These are the most hazardous materials within their class. For flammable liquids (Class 3), PG I materials have an initial boiling point of 95 degrees Fahrenheit (35 degrees Celsius) or below — meaning they vaporize very easily and create an explosive atmosphere fast.
- **Example**: Diethyl ether (UN1155, Class 3, PG I) — extremely volatile, flash point of -49 degrees Fahrenheit. A single spilled drum can create a vapor cloud that reaches an ignition source dozens of feet away.

### Packing Group II — Medium Danger
Mid-range hazard. For flammable liquids, PG II materials have a flash point below 73 degrees Fahrenheit (23 degrees Celsius) and an initial boiling point above 95 degrees Fahrenheit.
- **Example**: Gasoline (UN1203, Class 3, PG II) — flammable and volatile, but not as extreme as PG I materials.

### Packing Group III — Minor Danger
The lowest hazard level within a class, though these materials are still regulated for a reason. For flammable liquids, PG III materials have a flash point between 73 degrees Fahrenheit and 141 degrees Fahrenheit (23-60 degrees Celsius).
- **Example**: Diesel fuel (NA1993, Class 3, PG III) — combustible, but significantly harder to ignite than gasoline. It requires sustained heat to reach its flash point.

### Why Packing Groups Matter for Fleet Operations

Packing groups affect more than just the container the material ships in. They influence:

- **Packaging strength and testing requirements**: PG I containers must pass more rigorous drop tests and pressure tests.
- **Quantity limitations**: Some modes of transport restrict PG I materials to smaller quantities.
- **Placarding thresholds**: Certain placarding rules apply differently based on packing group (Table 2 vs. Table 1 materials under 49 CFR 172.504).
- **Segregation rules**: PG I materials may have additional separation requirements during transport.

A driver accepting a shipment should note the packing group on the shipping papers. A PG I flammable liquid demands more caution than a PG III flammable liquid — not because the regulations say to drive more carefully, but because the material itself is more dangerous if something goes wrong.

## Section 6: Common Mistakes

Misusing the Hazmat Table leads to violations, failed inspections, and unsafe shipments. Here are the errors that fleet operators, shippers, and drivers make most often.

### Mistake 1: Using the Wrong Proper Shipping Name
Trade names, abbreviations, and common names are not acceptable on shipping papers. "Gas" is not a proper shipping name. "MEK" is not a proper shipping name. The material must be listed by its exact proper shipping name as it appears in Column 2 of the Hazmat Table. If a customer calls it "thinning solvent," you still need to look it up and use the correct proper shipping name.

### Mistake 2: Missing or Wrong Hazard Class
Every hazardous material has a specific hazard class. Using the wrong class means wrong labels, wrong placards, and wrong emergency response instructions. A material classified as Class 8 (Corrosive) that is actually Class 6.1 (Toxic) will get the wrong treatment in an emergency.

### Mistake 3: Ignoring Special Provisions
Special provisions can exempt certain forms or concentrations of a material from the HMR entirely, or they can add requirements beyond the standard rules. Skipping Column 6 means missing critical information. For example, Special Provision 148 exempts certain concentrations of hydrogen peroxide from the HMR. If you do not check, you might be packaging and documenting a non-regulated material as hazmat — wasting time and money — or worse, treating a regulated material as non-hazmat.

### Mistake 4: Ignoring the "G" Symbol
When a "G" appears in Column 1, the proper shipping name is generic (like "Corrosive liquid, n.o.s."). The shipper must add the technical name of the hazardous constituent in parentheses. Failing to include the technical name is a violation and deprives emergency responders of information they need.

### Mistake 5: Confusing NA and UN Numbers
"NA" numbers are valid only in the United States and Canada. For international shipments, a "UN" number must be used. Shipping a material internationally with an NA number is a violation that can cause rejection at borders or ports.

### Mistake 6: Assuming Two Materials with the Same Class Are Identical
Gasoline and acetone are both Class 3, PG II flammable liquids. But their packaging authorizations, special provisions, and emergency response procedures differ. Always look up the specific material — do not copy the information from a different entry with the same class and packing group.

## Key Takeaways
- The Hazmat Table at 49 CFR 172.101 is the master reference for all regulated hazardous materials
- Each entry provides the proper shipping name, hazard class, ID number, packing group, packaging rules, and quantity limits
- Always use the exact proper shipping name from the table — no trade names or abbreviations
- The nine hazard classes cover explosives, gases, flammable liquids, flammable solids, oxidizers, toxics, radioactive, corrosives, and miscellaneous
- Packing Groups I, II, and III indicate great, medium, and minor danger within a class
- Always check special provisions — they can change what is required for a specific material

## Assessment Questions

### Q1
**Type**: multiple_choice
**Question**: You need to prepare shipping papers for a load of gasoline. What is the correct proper shipping name, hazard class, and UN number?
**A**: Fuel, Class 3, UN1203
**B**: Gasoline, Class 3, UN1203
**C**: Gasoline, Class 2.1, UN1075
**D**: Petroleum Product, Class 3, NA1993
**Correct Answer**: B
**Explanation**: The Hazmat Table lists gasoline under the proper shipping name "Gasoline," hazard class 3 (Flammable Liquid), with identification number UN1203. "Fuel" and "Petroleum Product" are not proper shipping names. UN1075 is propane (a gas, not a liquid). NA1993 is the ID number for diesel fuel, not gasoline.
**CFR Reference**: 49 CFR 172.101

### Q2
**Type**: multiple_choice
**Question**: What does Packing Group I indicate about a hazardous material?
**A**: The material presents minor danger
**B**: The material presents medium danger
**C**: The material presents great danger
**D**: The material does not require packaging
**Correct Answer**: C
**Explanation**: Packing Group I indicates "great danger" — the highest level of hazard within a class. PG I materials require the most robust packaging and strictest handling. PG II is medium danger and PG III is minor danger.
**CFR Reference**: 49 CFR 172.101(f)

### Q3
**Type**: multiple_choice
**Question**: A "G" symbol in Column 1 of the Hazmat Table means:
**A**: The material is only regulated when transported as a gas
**B**: The entry is a generic or n.o.s. name and the technical name must be included on shipping papers
**C**: The material is exempt from placarding requirements
**D**: The material is classified as "Generally Recognized As Safe"
**Correct Answer**: B
**Explanation**: The "G" symbol indicates a generic proper shipping name such as "Flammable liquid, n.o.s." When this symbol appears, the shipper must add the technical name of the hazardous constituent(s) in parentheses on shipping papers and package markings. This ensures emergency responders know what the material actually is.
**CFR Reference**: 49 CFR 172.101(b)

### Q4
**Type**: true_false
**Question**: Diesel fuel and gasoline are both Class 3 flammable liquids, so they always share the same UN identification number and packing group.
**Correct Answer**: False
**Explanation**: Although both are Class 3 Flammable Liquids, they have different identification numbers (gasoline is UN1203, diesel fuel is NA1993) and different packing groups (gasoline is PG II, diesel fuel is PG III). Each material has its own entry in the Hazmat Table with specific requirements. Never assume two materials in the same class are interchangeable.
**CFR Reference**: 49 CFR 172.101

### Q5
**Type**: multiple_choice
**Question**: Which hazard class covers materials like lithium batteries, dry ice, and elevated temperature liquids?
**A**: Class 4 — Flammable Solids
**B**: Class 6 — Toxic Substances
**C**: Class 8 — Corrosive Materials
**D**: Class 9 — Miscellaneous Hazardous Materials
**Correct Answer**: D
**Explanation**: Class 9 is the miscellaneous hazardous materials class. It covers materials that present a hazard during transportation but do not meet the definitions of Classes 1 through 8. Lithium batteries (UN3481), dry ice/solid carbon dioxide (UN1845), and elevated temperature materials all fall under Class 9.
**CFR Reference**: 49 CFR 173.140

### Q6
**Type**: scenario
**Question**: A shipper hands you shipping papers for a load described as "Corrosive liquid, n.o.s., Class 8, UN1760, PG II." The Hazmat Table entry for UN1760 has a "G" in Column 1. What is wrong with this shipping paper description?
**A**: The proper shipping name should be "Corrosive liquid" without "n.o.s."
**B**: The packing group is wrong — it should be PG I
**C**: The technical name of the corrosive constituent must be included in parentheses after the proper shipping name
**D**: Nothing is wrong — the description is complete as written
**Correct Answer**: C
**Explanation**: When a "G" symbol appears in Column 1 of the Hazmat Table, the proper shipping name is generic and the shipper must include the technical name of the hazardous constituent(s) in parentheses. The correct description would be something like "Corrosive liquid, n.o.s. (contains hydrofluoric acid), Class 8, UN1760, PG II." Without the technical name, the entry is incomplete and violates 49 CFR 172.203(k).
**CFR Reference**: 49 CFR 172.203(k)

### Q7
**Type**: multiple_choice
**Question**: Columns 8A, 8B, and 8C of the Hazmat Table provide which type of information?
**A**: Emergency response guide numbers
**B**: Packaging authorization references for exceptions, non-bulk, and bulk
**C**: Quantity limitations for passenger and cargo aircraft
**D**: Special provision numbers that modify standard requirements
**Correct Answer**: B
**Explanation**: Columns 8A, 8B, and 8C reference the specific sections of 49 CFR Part 173 that authorize packagings. Column 8A covers packaging exceptions, 8B covers non-bulk packaging authorizations, and 8C covers bulk packaging authorizations. Quantity limitations are in Columns 9A and 9B, and special provisions are in Column 6.
**CFR Reference**: 49 CFR 172.101(a)
