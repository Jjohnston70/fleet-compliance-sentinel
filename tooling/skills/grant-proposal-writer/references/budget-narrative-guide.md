# Budget Narrative Writing Guide

How to build grant budgets that align with program activities and satisfy reviewer scrutiny.

## Budget Narrative Principles

Every line item must answer three questions:
1. **What is it?** (specific item or cost)
2. **Why do you need it?** (connection to program activity)
3. **How did you calculate the cost?** (show your math)

A budget narrative is not a list of numbers -- it's a justification document where every dollar is tied to a specific program activity described in the proposal narrative.

## Standard Federal Budget Categories

### Personnel

```
[Name or TBD], [Title] — [X% FTE / X hours per week]
Annual salary: $XX,XXX
Grant-funded portion: $XX,XXX (X% of salary)
Role: [2-3 sentences describing what this person does on the project,
tied to specific activities in the program design]

Calculation: $XX,XXX annual salary x X% FTE = $XX,XXX
```

**Tips:**
- Salary rates must be reasonable for the position, region, and organization
- If requesting a new position, benchmark salary to comparable roles
- Show total salary and the portion charged to this grant
- If staff work on multiple grants, FTE allocations must total ≤100%
- Include a brief role description that matches the program narrative

### Fringe Benefits

```
Fringe benefits are calculated at XX% of total personnel costs.
This rate is based on [organization's actual fringe rate / standard rate].

Components include:
- FICA (7.65%)
- Health insurance ($X,XXX per employee)
- Retirement (X% employer match)
- Workers' compensation (X%)
- Unemployment insurance (X%)

Total fringe: $XX,XXX x XX% = $XX,XXX
```

**Tips:**
- Use your organization's actual fringe rate if you have one
- If not, calculate based on actual benefits costs
- Rate should be consistent with what you report on other grants

### Travel

```
[Purpose of travel]: [Activity it supports]
- X trips x $XXX airfare/mileage = $X,XXX
- X nights x $XXX lodging (per GSA rate for [city]) = $X,XXX
- X days x $XX per diem (per GSA rate) = $XXX
- Ground transportation: $XXX
Total for this trip type: $X,XXX

Local travel:
- XX miles/month x $0.67/mile (IRS rate) x 12 months = $X,XXX
- Purpose: [Site visits, participant transportation, partner meetings]
```

**Tips:**
- Use GSA per diem rates for federal grants (www.gsa.gov)
- Justify the purpose of each trip
- Local mileage should reference current IRS rate
- No first-class travel on federal grants

### Equipment

```
[Item]: $X,XXX
Purpose: [How it supports program activities]
Justification: [Why this specific item is needed and this cost is reasonable]
```

**Tips:**
- Equipment = single items over $5,000 (federal definition; check NOFO)
- Requires specific justification per item
- Must explain why purchase is preferable to lease
- Equipment purchased with federal funds has disposition rules

### Supplies

```
Office supplies: $XXX/month x 12 months = $X,XXX
[Specific program supplies]: $X,XXX
  - [Item]: $XXX (for [activity])
  - [Item]: $XXX (for [activity])
Technology/software: $X,XXX
  - [Software license]: $XXX/year (for [purpose])
Participant materials: $X,XXX
  - [Materials]: $XXX x [# participants] = $X,XXX
```

**Tips:**
- Supplies = consumable items under $5,000
- Be specific rather than lumping into "office supplies"
- Tie supplies to activities

### Contractual / Subawards

```
[Contractor/Subrecipient name or TBD]: $XX,XXX
Scope: [What they will do]
Selection: [How they were/will be selected — competitive procurement if >$10K federal]
Rate: [Hourly rate or fixed fee, with justification]
Deliverables: [What the grant receives for this cost]
```

**Tips:**
- Distinguish contractors from subrecipients (different rules under 2 CFR 200)
- Competitive procurement required above micro-purchase threshold for federal
- Sole-source justification needed if no competition
- Include monitoring plan for subawards

### Other Direct Costs

```
[Cost]: $X,XXX
Purpose: [Activity it supports]
Calculation: [How the amount was determined]
```

Common "other" items: printing, postage, conference registration, participant incentives, background checks, insurance, rent allocation, utilities allocation.

**Tips:**
- Must be necessary, reasonable, and allocable to the project
- Participant incentives: check NOFO for limits and allowability
- Rent: only the portion attributable to the project

### Indirect Costs

```
Indirect cost rate: XX%
Applied to: Modified Total Direct Costs (MTDC) of $XX,XXX
Total indirect: $XX,XXX

[If NICRA:] Rate is based on Negotiated Indirect Cost Rate Agreement
with [cognizant agency], effective [date].

[If de minimis:] Organization has elected the 10% de minimis rate
per 2 CFR 200.414, applied to MTDC.
```

**Tips:**
- MTDC excludes: equipment, participant support costs, subawards over $25K, rent
- If you have a NICRA, use it; if not, you can use 10% de minimis
- Some NOFOs cap indirect costs — check before applying your full rate
- Foundation grants may not allow indirect at all (or limit to 10-15%)

## Budget-Narrative Alignment Checklist

Before finalizing, verify:

- [ ] Every activity in the program narrative has corresponding budget support
- [ ] Every budget line item ties to a described activity
- [ ] Personnel in the budget match personnel described in org capacity
- [ ] FTE allocations are realistic for the described workload
- [ ] Travel purposes connect to program activities
- [ ] Total budget falls within NOFO funding range
- [ ] Math is correct (line items sum to category totals, categories sum to total)
- [ ] Indirect cost rate is properly applied and documented
- [ ] Match/cost-share (if required) is identified with sources
- [ ] No unallowable costs included

## Foundation Budget Considerations

Community foundations typically want simpler budgets:

```
PROGRAM BUDGET

Revenue:
  [Funder name] grant request:     $XX,XXX
  Other grants:                     $XX,XXX
  Organization funds:               $XX,XXX
  In-kind contributions:            $XX,XXX
  Total revenue:                    $XX,XXX

Expenses:
  Personnel (X.X FTE):             $XX,XXX
  Program supplies and materials:   $X,XXX
  Participant support:              $X,XXX
  Travel (local):                   $X,XXX
  Technology/communications:        $X,XXX
  Evaluation:                       $X,XXX
  Administrative (XX%):             $X,XXX
  Total expenses:                   $XX,XXX
```

**Foundation tips:**
- Show the full program budget, not just what you're asking this funder for
- Demonstrate other revenue sources (foundations like to see diversification)
- Keep administrative costs reasonable (typically under 15-20%)
- If the foundation doesn't allow indirect, build admin costs into direct line items
- Be prepared to explain how foundation funds specifically will be used

## Match / Cost-Share Documentation

When match is required:

```
MATCH SOURCES

Cash Match:
  [Source]: $XX,XXX — [Status: confirmed/committed/pending]
  [Source]: $XX,XXX — [Status]
  Cash subtotal: $XX,XXX

In-Kind Match:
  Volunteer time: XX hours x $XX.XX/hr = $X,XXX
    [Description of volunteer activities]
    [Rate basis: BLS or Independent Sector rate]
  Donated space: XX sq ft x $XX/sq ft/month x 12 = $X,XXX
    [Description and fair market value basis]
  Donated equipment/supplies: $X,XXX
    [Description and valuation method]
  In-kind subtotal: $XX,XXX

Total Match: $XX,XXX (XX% of federal request)
Required Match: $XX,XXX (XX%)
Status: [Meets/Exceeds/Below requirement]
```

**Tips:**
- Don't count the same funds as match for multiple grants
- In-kind must be valued at fair market value
- Volunteer time rates should reference Independent Sector or BLS data
- Match sources should be confirmed, not speculative
