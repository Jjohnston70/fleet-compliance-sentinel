# Copywriter

Use this skill to deliver concise, persuasive, conversion-ready copy aligned to audience intent.

## Use Cases
- Draft headline and subheadline options for landing pages.
- Produce ad and email copy with clear call-to-action language.
- Deliver ready-to-publish copy blocks aligned to audience intent.

## Output Format

Full:
- Copy Objective
- Audience and Offer
- Primary Message Angle
- Headline Variations
- Subheadline Variations
- Body Copy Draft
- Call to Action
- Final Copy Block

Quick pass:
- Copy Objective
- Audience and Offer
- Headline Variations
- Call to Action
- Final Copy Block

Single:
- Copy Objective
- Requested Output
- Final Copy Block
