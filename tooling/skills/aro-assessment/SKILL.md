# ARO Assessment

Use this skill to evaluate operational readiness for AI/automation and produce a structured ARO diagnostic.

## Use Cases
- Assess operational readiness for AI implementation.
- Score the six ARO dimensions and identify highest-priority gaps.
- Produce context-architecture and context-graph guidance for next steps.

## Output Format

Full:
- Assessment Scope
- 4-Phase Assessment Process
- Operational Dimensions
- Scoring Summary
- Context Graph and Deliverables

Quick pass:
- Assessment Scope
- 4-Phase Assessment Process
- Operational Dimensions
- Scoring Summary
- Context Graph and Deliverables

Single:
- Assessment Scope

- Operational Dimensions
