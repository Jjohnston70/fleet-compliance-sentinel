# Common Grant Proposal Rejection Reasons

This reference catalogs the most frequent reasons grant proposals are rejected, organized by category. Use this as a checklist during the validation phase (Step 4, Level 2) to catch problems before funders do.

## Disqualifying Errors {#disqualifying}

These cause immediate rejection regardless of proposal quality. Check first.

| Error | What to Look For | Severity |
|-------|-----------------|----------|
| **Late submission** | Proposal submitted after deadline | FATAL |
| **Ineligible applicant** | Org type, geography, or certification doesn't meet eligibility | FATAL |
| **Missing required sections** | NOFO specifies sections; one or more is absent | FATAL |
| **Exceeds page/word limit** | Even by one page — reviewers may stop reading | FATAL |
| **Wrong format** | NOFO specifies format (font, margins, file type) and it's not followed | FATAL |
| **Missing required forms** | SF-424, budget forms, certifications not included | FATAL |
| **Missing required attachments** | Letters of support, resumes, MOUs, org chart not included | FATAL |
| **SAM.gov registration expired** | Federal grants require active registration at time of submission | FATAL |
| **Requesting wrong amount** | Amount requested outside the stated funding range | FATAL |

**Check these first. Everything else is irrelevant if any of these apply.**

## Weak Problem Statement {#problem-statement}

| Issue | Description | Severity |
|-------|-------------|----------|
| **No data** | Need is described anecdotally without statistics or research | HIGH |
| **Wrong data** | National statistics used without local context | HIGH |
| **Outdated data** | Statistics more than 3-5 years old without justification | MEDIUM |
| **No target population specificity** | "Veterans" instead of "transitioning veterans ages 25-44 in El Paso County" | HIGH |
| **Circular reasoning** | "We need funding because we don't have a program" (need defined by absence of solution) | HIGH |
| **Problem too broad** | Trying to solve everything instead of a specific, addressable problem | MEDIUM |
| **No connection to funder priorities** | Problem is real but doesn't align with what this funder cares about | HIGH |
| **Deficit framing only** | Describes community only by what's wrong, not strengths and assets | MEDIUM |

## Program Design Failures {#program-design}

| Issue | Description | Severity |
|-------|-------------|----------|
| **No logic model** | Activities don't connect logically to stated outcomes | HIGH |
| **Broken causal chain** | Activities described won't plausibly produce the claimed outcomes | HIGH |
| **Vague activities** | "Provide mentorship" without describing what that looks like operationally | HIGH |
| **Unmeasurable outcomes** | "Increase awareness" or "build resilience" without defining measurable indicators | HIGH |
| **Unrealistic timeline** | Too much proposed in too short a timeframe | MEDIUM |
| **No implementation detail** | Grand vision but no operational plan for how activities actually happen | HIGH |
| **Duplicating existing services** | Proposing something that already exists locally without explaining differentiation | MEDIUM |
| **Missing dosage/intensity** | Doesn't describe how much service each participant receives | MEDIUM |
| **No recruitment strategy** | Assumes participants will show up without describing how they'll be reached | MEDIUM |
| **Ignoring barriers** | Doesn't acknowledge or address barriers to participation (transportation, childcare, stigma) | MEDIUM |

## Budget Issues {#budget-issues}

| Issue | Description | Severity |
|-------|-------------|----------|
| **Math errors** | Numbers don't add up (surprisingly common) | HIGH |
| **Budget-narrative disconnect** | Budget doesn't reflect the activities described in the narrative | HIGH |
| **No justification** | Line items without explanation of how costs were calculated | HIGH |
| **Unreasonable costs** | Salaries too high or too low for the position and region | MEDIUM |
| **Missing key costs** | Program clearly needs something (travel, supplies, technology) but it's not budgeted | MEDIUM |
| **Over-reliance on grant** | 100% of program funded by this grant — no diversification | MEDIUM |
| **Supplanting** | Using grant funds to pay for something currently funded by other sources | HIGH |
| **Indirect cost errors** | Wrong rate applied, or no NICRA when rate exceeds de minimis 10% | MEDIUM |
| **Cost-share not documented** | Match required but sources not identified or confirmed | HIGH |
| **Personnel over-allocated** | Key staff at 150% time across all funding sources | MEDIUM |

## Organizational Capacity Gaps {#org-capacity}

| Issue | Description | Severity |
|-------|-------------|----------|
| **No track record** | Organization has never run a similar program | MEDIUM |
| **Key personnel missing** | Critical roles (Program Director, Evaluator) not identified | HIGH |
| **Unqualified key personnel** | Staff don't have credentials/experience for their proposed roles | MEDIUM |
| **Too much for org size** | Small org proposing program that would double their budget/staff | HIGH |
| **No financial management** | No description of financial oversight, accounting, or audit history | MEDIUM |
| **Board not engaged** | No evidence of board involvement in or approval of the proposed program | MEDIUM |
| **No relevant partnerships** | Program requires community collaboration but no MOUs or letters of support | MEDIUM |

## Evaluation Plan Weaknesses {#evaluation}

| Issue | Description | Severity |
|-------|-------------|----------|
| **No evaluation plan** | Completely absent | HIGH |
| **Outputs confused with outcomes** | "Number of workshops held" is an output, not an outcome | HIGH |
| **No baseline** | How will you measure change without knowing where you started? | MEDIUM |
| **No data collection plan** | Says they'll measure things but doesn't describe how data will be collected | MEDIUM |
| **No comparison or benchmarks** | No way to know if results are good or bad without reference points | LOW |
| **Evaluation not independent** | Program staff evaluating their own program with no external validation | LOW |
| **No feedback loops** | No plan for using evaluation data to improve the program | MEDIUM |
| **Aspirational metrics** | Outcomes that can't realistically be achieved in the grant period | MEDIUM |

## Writing and Presentation Issues {#writing}

| Issue | Description | Severity |
|-------|-------------|----------|
| **Jargon overload** | Too much insider language without definition | MEDIUM |
| **Recycled boilerplate** | Clearly copy-pasted from another proposal; references wrong funder or program | HIGH |
| **Internal inconsistencies** | Numbers or claims in one section contradict another | HIGH |
| **No specificity** | Everything is vague — "many," "significant," "various" instead of actual numbers | HIGH |
| **Emotional appeal without evidence** | Moving stories without data to support the need | MEDIUM |
| **Too long/too detailed** | Overwhelms reviewers with unnecessary detail | LOW |
| **Poor organization** | Doesn't follow the order or structure the NOFO specifies | MEDIUM |
| **Typos and formatting errors** | Signals lack of attention to detail | LOW |

## Sustainability Concerns {#sustainability}

| Issue | Description | Severity |
|-------|-------------|----------|
| **No sustainability plan** | What happens when this grant ends? Silence. | HIGH |
| **"We'll apply for more grants"** | Not a sustainability plan — it's a dependency plan | MEDIUM |
| **No revenue diversification** | No other funding sources described | MEDIUM |
| **No institutionalization path** | No plan for how the program becomes embedded in ongoing operations | MEDIUM |
| **Unrealistic sustainability claims** | "Program will be self-sustaining in year 2" with no explanation of how | MEDIUM |

---

## Alignment Failures {#alignment}

| Issue | Description | Severity |
|-------|-------------|----------|
| **Wrong funder** | Proposal doesn't match funder's mission, priorities, or geographic focus | HIGH |
| **Ignoring NOFO priorities** | NOFO lists specific priorities but proposal doesn't address them | HIGH |
| **Mismatched population** | Funder targets one population but proposal serves a different one | HIGH |
| **Scale mismatch** | Local funder getting a national-scope proposal, or vice versa | MEDIUM |
| **Innovation claims without evidence** | Says "innovative" but describes standard practice | LOW |
| **No awareness of funder context** | Doesn't reference funder's strategic plan, previous investments, or stated interests | MEDIUM |

## How to Use This Reference

During validation (Step 4, Level 2):

1. Go through each category above
2. For each issue, check whether it applies to the proposal being evaluated
3. If an issue is found:
   - Note the specific evidence (page, section, or quote from proposal)
   - Rate severity: HIGH, MEDIUM, or LOW
   - Write a specific recommendation for how to fix it
4. Compile all findings into the "Common Rejection Risk Analysis" section of the evaluation report

Priority the fix recommendations:
- **Must Fix (HIGH)**: Would likely cause rejection or significant score reduction
- **Should Fix (MEDIUM)**: Weakens competitiveness; easy points left on the table
- **Nice to Fix (LOW)**: Polish items; address if time permits before deadline
