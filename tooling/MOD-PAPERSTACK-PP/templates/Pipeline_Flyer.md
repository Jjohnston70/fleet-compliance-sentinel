<!--
  Pipeline Flyer - MARKDOWN VERSION
  TRUE NORTH DATA STRATEGIES
  =============================================

  HOW TO USE:
  1. Save this file as Pipeline_Flyer.md
  2. Open in any Markdown viewer: VS Code preview (Ctrl+Shift+V), GitHub, Notion, Obsidian, etc.
  3. Edit text directly - Markdown is just plain text with simple formatting symbols
  4. No special software needed to edit - any text editor works

  MARKDOWN CHEAT SHEET:
  - # Heading 1          = Biggest heading (like <h1> in HTML)
  - ## Heading 2         = Second level heading
  - ### Heading 3        = Third level heading
  - **bold text**        = Bold
  - *italic text*        = Italic
  - ***bold italic***    = Both bold and italic
  - - item               = Bullet list item
  - 1. item              = Numbered list item
  - [link text](url)     = Clickable link
  - > quote              = Block quote (indented text)
  - ---                  = Horizontal line/divider
  - `code`               = Inline code (monospace)
  - | col1 | col2 |      = Table (see below for full syntax)
  - &nbsp;               = Non-breaking space (forces a space)
  - <!-- comment -->     = Hidden comment (only visible in source, not rendered)

  TABLE SYNTAX:
  | Header 1 | Header 2 |        <-- column headers
  |----------|----------|        <-- separator row (required)
  | Cell 1   | Cell 2   |        <-- data rows
  | :---     | centered |        <-- colon on left = left-align
  | :---:    | centered |        <-- colons on both sides = center
  | ---:     | right    |        <-- colon on right = right-align

  NOTES ON MARKDOWN LIMITATIONS:
  - Markdown has NO background colors, NO custom fonts, NO CSS styling
  - It renders differently depending on the viewer (GitHub vs VS Code vs Notion)
  - It's meant for CONTENT first, not pixel-perfect design
  - For styled/branded output, use the HTML or PDF versions instead
  - This version is best for: README files, GitHub pages, Notion imports, quick sharing
-->
