FastAPI Conversion Guide

To convert this prototype to a real FastAPI + HTMX app:

command-center/
├── main.py                    # FastAPI app + routes
├── models.py                  # Pydantic models
├── database.py                # DB connection
├── templates/
│   ├── base.html              # Shell layout (this file's HTML)
│   ├── dashboard.html         # Dashboard panel
│   ├── roster.html            # Roster panel
│   └── partials/
│       ├── activity-feed.html # Recent activity table rows
│       ├── notes.html         # Single note item
│       ├── member-row.html    # Roster table row
│       ├── member-edit.html   # Inline edit form
│       ├── grant-detail.html  # Grant modal content
│       ├── status-card.html   # Buddy Backup polling target
│       └── pagination.html    # Page nav
└── static/
    └── styles.css             # This file's CSS

Each partial returns a fragment of HTML — not a full page. HTMX swaps it into the target element. The server always controls the data and rendering. The browser just displays it.
