# Notes

This folder is your lab notebook. Iteration, thinking, AI output drafts.

## What Goes Here

- Session notes: `session-YYYY-MM-DD.md`
- Research drafts: `research-topic.md`
- TODO lists: `todo.md`
- AI-generated exploration
- Improvement plans in progress

## What Does NOT Go Here

- Final documentation → `/docs`
- Executable scripts → `/tools`
- Frozen experiments → `/archive`

## Naming Convention

Use lowercase with dates or descriptive slugs:
- `session-2025-01-15.md`
- `research-authentication.md`
- `todo.md`
- `improvement-plan-draft.md`

## Lifecycle

Notes can be:
1. **Promoted** to `/docs` when finalized
2. **Archived** to `/archive` when no longer relevant
3. **Deleted** when truly disposable
