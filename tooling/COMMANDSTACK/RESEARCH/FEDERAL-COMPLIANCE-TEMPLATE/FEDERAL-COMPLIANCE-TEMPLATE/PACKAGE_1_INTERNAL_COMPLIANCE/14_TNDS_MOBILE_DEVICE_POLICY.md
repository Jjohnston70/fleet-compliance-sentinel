TRUE NORTH DATA STRATEGIES LLC
MOBILE DEVICE POLICY

Purpose

This Mobile Device Policy establishes True North Data Strategies LLC's requirements for the secure use of mobile devices to access company systems, applications, and data. This policy ensures mobile devices are properly secured, managed, and used in compliance with security standards.

Scope

This policy applies to all True North Data Strategies personnel, contractors, subcontractors, and any other individuals using mobile devices to access company systems or data. It covers company-issued mobile devices and personal mobile devices used for business purposes (BYOD).

Governance and Cross-Package References

This policy operates under the authority of the Security and Compliance Handbook (Package 2), which establishes the master governance framework for mobile device security.

Universal TNDS Principles:
Data-as-Regulated: TNDS treats all client and customer data as regulated by default; mobile devices must protect data with equivalent security to fixed systems
No Sensitive Data in Logs: Mobile device management logs and monitoring must not contain PII, PHI, PCI, CUI, or other regulated identifiers in unprotected form

Cross-Package Integration:
Package 2 (Security & Compliance Handbook): Master governance document establishing mobile device security requirements
Package 1 (BYOD Policy): Detailed requirements for personal mobile devices used for business
Package 1 (Remote Access Policy): Remote access requirements applicable to mobile devices
Package 7 (Mobile Device Security Policy): Advanced mobile device security requirements

For BYOD requirements, see Package 1 (BYOD Policy).
For remote access from mobile devices, see Package 1 (Remote Access Policy).
For advanced mobile security, see Package 7 (Mobile Device Security Policy).
