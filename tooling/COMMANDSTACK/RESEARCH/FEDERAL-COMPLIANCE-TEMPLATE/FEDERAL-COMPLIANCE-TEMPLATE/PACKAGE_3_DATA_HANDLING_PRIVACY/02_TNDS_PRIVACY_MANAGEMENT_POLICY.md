TRUE NORTH DATA STRATEGIES LLC
PRIVACY MANAGEMENT POLICY

Purpose

This Privacy Management Policy establishes True North Data Strategies LLC's framework for protecting the privacy of individuals whose personal information we collect, process, store, or share. This policy ensures compliance with applicable privacy laws and regulations and demonstrates our commitment to privacy protection.

Scope

This policy applies to all True North Data Strategies personnel, contractors, subcontractors, systems, and processes that involve personal information. It covers privacy program governance, privacy impact assessments, consent management, individual rights management, and privacy compliance monitoring.

Governance and Cross-Package References

This policy establishes the privacy management framework that governs all TNDS operations involving personal information.

Universal TNDS Principles:
Data-as-Regulated: TNDS treats all client and customer data as regulated by default, ensuring maximum privacy protection
No Sensitive Data in Logs: Privacy management documentation must not contain PII, PHI, PCI, CUI, or other regulated identifiers in unprotected form

Cross-Package Integration:
Package 2 (Security & Compliance Handbook): Master governance document; this policy implements privacy requirements
Package 3 (Data Handling Standard): Authoritative source for data classification and handling
Package 3 (Anonymization Standard): Provides techniques for privacy protection
Package 1 (Internal Compliance): All Package 1 policies defer to Package 3 for privacy requirements

For data handling requirements, see Package 3 (Data Handling Standard).
For anonymization techniques, see Package 3 (Anonymization Standard).

Individual Rights Management

Individual Rights

Right to Access
Individuals can request access to their personal information
Access provided within regulatory timeframes (typically 30-45 days)
Access includes copy of data and processing information
Access provided in commonly used format
Access free of charge (with exceptions for excessive requests)

Right to Rectification
Individuals can request correction of inaccurate personal information
Corrections made promptly
Corrected data communicated to recipients if applicable
Correction requests documented

Right to Erasure (Right to be Forgotten)
Individuals can request deletion of their personal information
Deletion performed when legally required
Exceptions for legal obligations, public interest, legal claims
Deletion communicated to recipients if applicable
Deletion requests documented

Right to Restrict Processing
Individuals can request limitation of processing
Processing restricted while accuracy or lawfulness verified
Restricted data marked and processing limited
Restriction lifted when conditions resolved
Restriction requests documented

Right to Data Portability
Individuals can request their data in portable format
Data provided in structured, commonly used, machine-readable format
Data transmitted to another controller if technically feasible
Portability applies to data provided by individual and processed by automated means

Right to Object
Individuals can object to processing based on legitimate interests
Individuals can object to direct marketing
Individuals can object to automated decision-making
Objections assessed and processing stopped unless compelling legitimate grounds
Objection requests documented

Right to Opt-Out (CCPA/CPRA)
California residents can opt-out of sale or sharing of personal information
Opt-out honored within 15 days
Opt-out preference signal respected
Opt-out not required for service
Opt-out requests documented
