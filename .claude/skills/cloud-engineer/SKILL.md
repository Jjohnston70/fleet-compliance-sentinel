# Cloud Engineer

Use this skill for implementation-grade cloud architecture, deployment planning, and operational hardening.

## Use Cases
- Design secure cloud architecture for new workloads.
- Plan infrastructure as code rollout and deployment sequencing.
- Optimize cloud cost and reliability with concrete implementation steps.

- Verification and Operations
