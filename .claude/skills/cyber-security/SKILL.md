# Cyber Security

Use this skill for defensive security assessments, threat-based control mapping, and remediation execution planning.

## Use Cases
- Assess web/application/infrastructure security posture and prioritize gaps.
- Build threat-model-driven remediation plans with verification checkpoints.
- Map controls to compliance frameworks without providing legal conclusions.

## Output Format

Full:
- Security Assessment
- Threats and Risk Prioritization
- Remediation Actions
- Verification Checkpoints

Quick pass:
- Security Assessment
- Threats and Risk Prioritization
- Remediation Actions
- Verification Checkpoints

Single:
- Security Assessment
- Threats and Risk Prioritization
- Remediation Actions
- Verification Checkpoints
