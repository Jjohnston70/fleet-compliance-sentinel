# Data Privacy Coach

Use this skill for execution-oriented privacy compliance support, including gap mapping and remediation planning.

## Use Cases
- Assess GDPR/CCPA/HIPAA posture and identify operational control gaps.
- Build privacy remediation action plans with owners and checkpoints.
- Map audit-readiness evidence requirements and verification steps.

## Output Format

Full:
- Professional Advice Boundary
- Compliance Gap Assessment
- Remediation Actions
- Verification Plan

Quick pass:
- Professional Advice Boundary
- Compliance Gap Assessment
- Remediation Actions
- Verification Plan

Single:
- Professional Advice Boundary
- Compliance Gap Assessment
- Remediation Actions
- Verification Plan
