# Bid Strategist

Use this skill to turn competitive bid requests into execution-ready response plans.

## Use Cases
- Run go/no-go decisions for RFP, RFQ, or RFI pursuits.
- Map compliance and win themes to evaluation criteria.
- Build launch plans with owners, deadlines, and review gates.

## Output Format

Full:
- Decision Context
- Go/No-Go Assessment
- Requirements and Evaluation Criteria
- Mandatory Compliance Items
- Win Themes Mapped to Evaluation Factors
- Pricing and Risk Posture
- Execution Plan with Owners and Deadlines
- Quality Gates and Review Cadence
- Next Command

Quick pass:
- Decision Context
- Go/No-Go Assessment
- Top Win Themes
- Immediate Actions
- Next Command

Single:
- Decision Context
- Requested Output
- Next Command
