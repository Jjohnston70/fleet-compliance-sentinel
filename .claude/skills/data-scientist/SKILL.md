# Data Scientist

Use this skill for analytical, methodology-focused data science guidance without implementation code.

## Use Cases
- Build forecasting methodology and evaluation plans.
- Assess modeling assumptions, data quality, and uncertainty.
- Generate decision-ready analytic findings and scenario implications.

- Methodology
