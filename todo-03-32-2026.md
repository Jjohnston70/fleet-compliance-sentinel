2. Gateway supports `node` and `python` module adapters.
3. Add phase-1 invokers for:
   - `ML-EIA-PETROLEUM-INTEL` (pipeline + report export)
   - `ML-SIGNAL-STACK-TNCC` (source pipeline run)
   - `MOD-PAPERSTACK-PP` (document generation actions)
